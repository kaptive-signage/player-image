= Kaptive player image

Small image for Raspberry Pi 5 with the kaptive player.

== Build

To start the docker environment needed to build the image, run:

```bash
docker compose build
docker compose run --rm custom-rpi-image
```

This image is build using the `rpi-image-gen` tool. To build the staging image run:

```bash
rpi-image-gen/build.sh -o ~/kaptive-arm64/kaptive-staging.options -D ~/kaptive-arm64/ -c kaptive && cp /home/imagegen/rpi-image-gen/work/kaptive-player-staging-arm64/deploy/* /tmp/kaptive-player-arm64/out
```

To build the demo image run:

```bash
rpi-image-gen/build.sh -o ~/kaptive-arm64/kaptive-demo.options -D ~/kaptive-arm64/ -c kaptive && cp /home/imagegen/rpi-image-gen/work/kaptive-player-demo-arm64/deploy/* /tmp/kaptive-player-arm64/out
```
